## Цель
Добавить в шапку сайта ссылку / кнопку «Админ-панель» для быстрого перехода на `/admin`.

## Проблема
Сейчас шапка (`<header>`) дублируется в двух публичных страницах: `src/routes/index.tsx` и `src/routes/games.tsx`. В админ-панели (`src/routes/admin.tsx`) своя отдельная шапка.

## План
1. **Создать единый компонент `Header`** (`src/components/Header.tsx`), объединяющий текущие две шапки.
2. **Добавить ссылку «Админ-панель»** в навигацию компонента `Header` через `<Link to="/admin">`.
3. **Заменить inline-шапки** в `src/routes/index.tsx` и `src/routes/games.tsx` на новый компонент `<Header />`.
4. **Проверить адаптивность**: ссылка должна корректно отображаться в десктопной навигации и мобильном меню (бургер).

## Технические детали
- Использовать `<Link>` из `@tanstack/react-router` для типобезопасной навигации.
- Стиль ссылки — стандартный пункт навигации (`text-sm font-medium text-muted-foreground hover:text-foreground`), чтобы не выбиваться из дизайна.
- Для мобильного меню (`menuOpen` на главной) добавить пункт «Админ-панель» в выпадающий список.
