## 3 ключевых изменения для идеального вида

### 1. Единая ритмика отступов (вертикальный ритм)
Сейчас секции используют `py-24` без учёта мобильного экрана — на телефоне всё слишком разрежено, а заголовки прижаты к контенту. Приведу к единой шкале:
- Секции: `py-16 sm:py-20 lg:py-28` вместо `py-24`
- Между заголовком секции и контентом: фиксированно `mt-10 sm:mt-14` (сейчас `mt-12` где-то есть, где-то нет)
- Карточки приставок: внутренний паддинг `p-5 sm:p-6`, gap сетки `gap-5 sm:gap-8`
- Сетка игр: `gap-3 sm:gap-4` (на мобильном 4px тесновато выглядят стикеры)
- Hero: `pt-28 sm:pt-32 pb-16 sm:pb-24` и `mt-6 sm:mt-8` для подзаголовка вместо `mt-8`

### 2. Плавные и мгновенные микро-взаимодействия
Сейчас часть hover/active использует `transition` (по умолчанию 150ms всех свойств) — иногда заметна "ватность", особенно на цвете. Унифицирую:
- Все интерактивные элементы: `transition-[transform,box-shadow,filter,background-color,border-color] duration-200 ease-out` (вместо общего `transition`)
- Active-состояние кнопок: оставляю `active:scale-[0.96]` (сейчас `0.92` — выглядит "проседающе"), `duration-100` для тактильного отклика
- Карточки приставок/шагов: hover поднимаются `hover:-translate-y-1` + мягкая тень `hover:shadow-[0_10px_30px_-10px_hsl(var(--primary)/0.35)]`
- Карточки игр: hover лёгкий `scale-[1.02]`, изображение `scale-105` уже есть — добавлю `duration-300 ease-out` для естественности
- FAQ-аккордеон: использует max-height — заменю на плавный CSS grid-rows трюк (`grid-template-rows: 0fr → 1fr`) для мгновенного открытия без задержки

### 3. Профессиональная типографика и выравнивание
- Заголовки секций: добавлю единообразный отступ снизу к описанию (`mt-3` вместо `mt-2`), вес `font-semibold` у подписи
- Стикеры на карточках игр: размер уже подняли — выровняю позиционирование (`top-2 left-2 gap-1.5`) и порядок (новинка → хит → 2🎮 уже на бэке)
- Цены: `tabular-nums` для ровных цифр
- Кнопки: одинаковая высота через `h-11` (вместо `py-3`/`py-4` вразнобой), letter-spacing согласован
- Карточки шагов "Как это работает": номер `01-04` и иконка визуально соревнуются — уменьшу номер до `text-4xl`, иконка `w-7 h-7`, добавлю `text-primary/60` для иконки чтобы цвет вёл номер
